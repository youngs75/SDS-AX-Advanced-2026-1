```
youngs75_a2a/                  ← 독립 레포 분리 가능
├── .env                       ← 메인 환경변수 파일
├── core/                      ← 공통 프레임워크
├── agents/                    ← 에이전트 4개 (simple_react, deep_research, orchestrator, coding_assistant)
├── a2a/                       ← A2A 프로토콜
├── eval_pipeline/             ← DeepEval 평가 파이프라인 (loop1~3, observability)
├── data/                      ← Golden Dataset, corpus, eval_results
├── scripts/                   ← 파이프라인 스크립트 01~10
├── tests/                     ← A2A 테스트 + eval/ (DeepEval 테스트)
├── docker/                    ← Docker Compose Harness (harness.yml, langfuse.yaml)
└── utils/                     ← 유틸리티
```
