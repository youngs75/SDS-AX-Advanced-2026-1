### 4. CodingAssistant → Coding Agent Harness 전환 (핵심)
- **code_tools MCP 서버 신규 생성** (`mcp_servers/code_tools/server.py`)
  - 5개 도구: read_file, write_file, list_directory, search_code, run_python
  - workspace 경계 안전 검증 (`_safe_path`)
- **에이전트 그래프 ReAct 루프 전환**
  - 이전: parse → execute(LLM만) → verify
  - 이후: parse → execute(LLM+MCP) ⇄ tools → verify → retry/END
  - `_should_use_tools()` 라우팅 — tool_calls 유무에 따라 분기
- **CodingConfig**: `mcp_servers`, `max_tool_calls` 필드 추가
- **CodingState**: `project_context`, `language` 필드 추가
- **프롬프트**: 프로젝트 구조 가이드 (`youngs75_a2a/` 경로 안내) + 도구 설명 주입
- **my_agent.py**: `await CodingAssistantAgent.create()` 비동기 초기화 패턴 적용
